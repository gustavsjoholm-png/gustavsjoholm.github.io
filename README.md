# Gustav Sjöholm - Journalist Portfolio

Personlig portfolio-hemsida för Gustav Sjöholm, journalist på TT Nyhetsbyrån.

## Om

En clean och professionell portfolio som presenterar mig som journalist, med utvalda artiklar och kontaktinformation.

## Features

- Responsiv design (fungerar på mobil och desktop)
- Profilbild
- Länkar till sociala medier (LinkedIn, X/Twitter)
- Bot-skyddad e-postkontakt
- Utvalda artiklar från TT Nyhetsbyrån

## Teknologi

- Ren HTML/CSS/JavaScript
- Ingen externa dependencies
- Hosting via GitHub Pages
